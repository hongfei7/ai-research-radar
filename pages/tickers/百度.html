Code Arena放榜：阿里千问3.7编程能力超越GPT、Gemini，仅次于Claude系列

阿里千问Qwen3.7-Max在Code Arena全球编程榜单中以1541分排名全球第二，超越GPT-5.5、Gemini-3.5-Flash、GLM-5.1等模型，仅次于Claude系列，成为唯一突破1540分的国产大模型。该模型面向Agent设计，可独立完成专业团队2周工作量的复杂项目，并支持35小时、1000次工具调用的长程任务。Code Arena采用开发者盲测投票机制，被视为最具公信力的AI编程评测之一。千问3.7的突破标志着国产大模型正式跻身编程能力第一梯队，对全球AI竞争格局产生深远影响。

阿里云加速AI出海，发布全新海外AI产品官网Qwen Cloud
